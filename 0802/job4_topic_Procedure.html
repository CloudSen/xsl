Uploading of the Navigation database
更新导航数据库

NOTE:This procedure is for the FMGC1. For the FMGC2, which is identical, use the indications between the parentheses.

注意:本程序适用于FMGC1和FMGC2,对于FMGC2使用括号中的指示。

NOTE:The result may be different due to different software version of the PMAT2000 data loader.

注意:PMAT2000数据装载机软件包版本不同，显示结果可能不同。

NOTE:When the uploading of the FMGC is completed, you can use the XLOAD function to load the other FMGC (032-099 101-114 116-116 118-150 Ref.TASK. 22-70-00-610-009-A ).
当一台FMGC数据库更新完成后，可以使用XLOAD功能来更新另一台FMGC。 (032-099 101-114 116-116 118-150 参考.TASK. 22-70-00-610-009-A )。

NOTE:The uploading time for one full disk is twelve minutes approximately.

注意:一张磁盘的更新时间大约需要12分钟。

NOTE:After the uploading operations, you must do a check of navigation database version and date through MCDU.

注意:加装完成后，必须通过MCDU检查导航数据库版本和日期。
